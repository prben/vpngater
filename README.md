# vpngatefetcher

Fetches preferred OpenVPN host file from VPNGate project and configures the same for your machine.

## Usage of vpngate:
    -latency int
        Minimum latency of VPN (ms) (default 40)
    -minSpeed int
        Minimum speed of VPN (Mbps) (default 40)
    -openvpnBin string
        Mention path of OpenVPN binary (default "/usr/sbin/openvpn")
     -proto string
        Preferred Protocol - tcp/udp/any (default "any")
